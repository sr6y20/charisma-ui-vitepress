Message 消息提示 ​

在界面中展示全局提示信息。

基础用法 ​

最简单的消息提示。

<template>
  <ChButton @click="open">显示消息</ChButton>
</template>

<script setup lang="ts">
import { ChMessage } from "@/components/Message/method";

const open = () => {
  ChMessage({ message: "这是一条消息提示" });
};
</script>

不同类型 ​

支持 info、success、warning、error 四种类型。

<template>
  <div class="ch-row">
    <ChButton @click="() => open('info')">Info</ChButton>
    <ChButton type="success" @click="() => open('success')">Success</ChButton>
    <ChButton type="warning" @click="() => open('warning')">Warning</ChButton>
    <ChButton type="danger" @click="() => open('error')">Error</ChButton>
  </div>
</template>

<script setup lang="ts">
import { ChMessage } from "@/components/Message/method";

const open = (type: "info" | "success" | "warning" | "error") => {
  ChMessage({ message: `这是一条 ${type} 消息`, type });
};
</script>

可关闭 ​

通过 showClose 显示关闭按钮。

<template>
  <ChButton @click="open">显示可关闭消息</ChButton>
</template>

<script setup lang="ts">
import { ChMessage } from "@/components/Message/method";

const open = () => {
  ChMessage({ message: "这是一条可关闭的消息", showClose: true });
};
</script>

自定义时长 ​

通过 duration 设置消息停留时间，单位为毫秒。设为 0 表示不会自动关闭。

<template>
  <div class="ch-row">
    <ChButton @click="open3000">3 秒关闭</ChButton>
    <ChButton @click="open0">不会自动关闭</ChButton>
  </div>
</template>

<script setup lang="ts">
import { ChMessage } from "@/components/Message/method";

const open3000 = () => {
  ChMessage({ message: "3 秒后自动关闭", duration: 3000 });
};

const open0 = () => {
  ChMessage({ message: "手动关闭", duration: 0, showClose: true });
};
</script>

多个消息 ​

多个消息会按顺序叠加。

<template>
  <ChButton @click="open">连续打开消息</ChButton>
</template>

<script setup lang="ts">
import { ChMessage } from "@/components/Message/method";

let count = 1;
const open = () => {
  ChMessage({ message: `第 ${count++} 条消息` });
};
</script>

API ​

Message(options) ​

参数说明类型默认值
message消息内容string | VNode-
type消息类型'info' | 'success' | 'warning' | 'error'info
duration停留时间（ms）number3000
showClose是否显示关闭按钮booleanfalse
offset距离顶部偏移量number20
