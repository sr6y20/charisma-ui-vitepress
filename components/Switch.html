Switch 开关 ​

在两种状态间切换时用到的开关选择器。

基础用法 ​

绑定 v-model 到一个 Boolean 类型的变量。可以使用 activeColor 属性与 inactiveColor 属性来设置开关的背景色。

<template>
  <div class="ch-row">
    <ChSwitch v-model="value1"></ChSwitch>
    <ChSwitch v-model="value2" activeColor="#8BC34A" inactiveColor="#D2120B"></ChSwitch>
  </div>
</template>

<script setup lang="ts">
import { ref } from "vue";

const value1 = ref(false);
const value2 = ref(false);
</script>

尺寸 ​

通过 size 属性设置开关的尺寸，可选值为 large、small。

<template>
  <div class="ch-row">
    <ChSwitch v-model="value1" size="large"></ChSwitch>
    <ChSwitch v-model="value2"></ChSwitch>
    <ChSwitch v-model="value3" size="small"></ChSwitch>
  </div>
</template>

<script setup lang="ts">
import { ref } from "vue";

const value1 = ref(false);
const value2 = ref(false);
const value3 = ref(false);
</script>

文字描述 ​

使用 active-text 属性与 inactive-text 属性来设置开关的文字描述。

关关关
N
✘
<template>
  <div class="ch-row">
    <ChSwitch v-model="value1" width="60" active-text="开开开" inactive-text="关关关"></ChSwitch>
    <ChSwitch v-model="value2" width="100" active-text="Y" inactive-text="N"></ChSwitch>
    <ChSwitch v-model="value3" active-text="✔" inactive-text="✘"></ChSwitch>
  </div>
</template>

<script setup lang="ts">
import { ref } from "vue";

const value1 = ref(false);
const value2 = ref(false);
const value3 = ref(false);
</script>

禁用状态 ​

设置 disabled 属性，接受一个 Boolean，设置 true 即可禁用。

<template>
  <div class="ch-row">
    <ChSwitch v-model="value1" disabled></ChSwitch>
    <ChSwitch v-model="value2" disabled></ChSwitch>
  </div>
</template>

<script setup lang="ts">
import { ref } from "vue";

const value1 = ref(true);
const value2 = ref(false);
</script>

Switch Props ​

参数说明类型默认值

inactiveText关闭时的文字描述string-

inactiveValue关闭时的值anyfalse
