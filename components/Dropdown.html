Dropdown 下拉菜单 ​

将动作或菜单折叠到下拉菜单中。

基础用法 ​

悬停在下拉菜单上以展开更多操作。

Dropdown List 
<template>
  <Dropdown :menuOptions="menuOptions">
    <div class="ch-row" style="cursor: pointer">
      <Text type="primary"
        >Dropdown List
        <Icon icon="angle-down"></Icon>
      </Text>
    </div>
  </Dropdown>
</template>

<script setup lang="ts">
import { ref } from "vue";
const menuOptions = ref([
  {
    label: "选项1",
    key: "1",
  },
  {
    label: "选项2",
    key: "2",
    disabled: true,
  },
  {
    label: "选项3",
    key: "3",
    divided: true,
  },
  {
    label: "选项4",
    key: "4",
  },
]);
</script>

可以配置点击激活或者悬停激活。

将 trigger 属性设置为 click 即可， 默认为 hover。

菜单隐藏方式 ​

可以通过 hide-after-click 属性来配置。

下拉菜单默认在点击菜单项后会被隐藏，将 hide-after-click 属性设置为 false 可以关闭此功能。

Dropdown List 
<template>
  <Dropdown :menuOptions="menuOptions" :hide-after-click="false">
    <div class="ch-row" style="cursor: pointer">
      <Text
        >Dropdown List
        <Icon icon="angle-down"></Icon>
      </Text>
    </div>
  </Dropdown>
</template>

<script setup lang="ts">
import { ref } from "vue";
const menuOptions = ref([
  {
    label: "选项1",
    key: "1",
  },

    key: "2",

    key: "3",
