Tooltip 文字提示 ​

当鼠标悬停在元素上时显示一段文字介绍。

基础用法 ​

在这里我们提供 12 种不同方向的展示方式，可以通过以下完整示例来理解，选择你要的效果。

<template>
  <div class="ch-row ch-row--center">
    <Tooltip content="Top Left prompts info" placement="top-start">
      <Button>top-start</Button>
    </Tooltip>
    <Tooltip content="Top prompts info" placement="top">
      <Button>top</Button>
    </Tooltip>
    <Tooltip content="Top End prompts info" placement="top-end">
      <Button>top-end</Button>
    </Tooltip>
  </div>

  <div class="ch-row ch-row--space-between">
    <Tooltip content="Left Start prompts info" placement="left-start">
      <Button>left-start</Button>
    </Tooltip>
    <Tooltip content="Right Start prompts info" placement="right-start">
      <Button>right-start</Button>
    </Tooltip>
  </div>

  <div class="ch-row ch-row--space-between">
    <Tooltip content="Left prompts info" placement="left">
      <Button>left</Button>
    </Tooltip>
    <Tooltip content="Right prompts info" placement="right">
      <Button>right</Button>
    </Tooltip>
  </div>

  <div class="ch-row ch-row--space-between">
    <Tooltip content="Left End prompts info" placement="left-end">
      <Button>left-end</Button>
    </Tooltip>
    <Tooltip content="Right End prompts info" placement="right-end">
      <Button>right-end</Button>
    </Tooltip>
  </div>

  <div class="ch-row ch-row--center">
    <Tooltip content="Bottom Left prompts info" placement="bottom-start">
      <Button>bottom-start</Button>
    </Tooltip>
    <Tooltip content="Bottom prompts info" placement="bottom">
      <Button>bottom</Button>
    </Tooltip>
    <Tooltip content="Bottom End prompts info" placement="bottom-end">
      <Button>bottom-end</Button>
    </Tooltip>
  </div>
</template>

主题 ​

Tooltip 组件内置了两个主题：dark 和 light

<template>
  <div class="ch-row">
    <Tooltip content="Top Center">
      <Button>Dark</Button>
    </Tooltip>

    <Tooltip content="Bottom Center" effect="light">
      <Button>Light</Button>
    </Tooltip>

    <Tooltip content="Bottom Center" effect="customized">
      <Button>Customized theme</Button>
    </Tooltip>
  </div>
</template>

更多内容的文字提示 ​

展示多行文本或者是设置文本内容的格式

用具名 slot content，替代 tooltip 中的 content 属性。

<template>
  <div class="ch-row">
    <Tooltip>
      <template #content> multiple lines<br />second line </template>
      <Button>Top</Button>
    </Tooltip>
  </div>
</template>

手动触发 ​

通过设置属性 manual 可以手动触发 tooltip，默认为 false。

<template>
  <div class="ch-row">
    <Tooltip
      :manual="manual"
      effect="light"
      placement="bottom"
      content="click to close tooltip function"
    >
      <Button @click="manual = !manual"
        >click to {{ manual ? "active" : "close" }} tooltip function</Button
      >
    </Tooltip>
  </div>
</template>

<script setup lang="ts">
import { ref } from "vue";

const manual = ref(false);
</script>

Attributes ​

参数说明类型默认值
content显示的内容，也可以通过 slot#content 传入 DOMstring-
manual手动触发booleanfalse
trigger触发方式stringhover
placementTooltip 的出现位置stringtop
popperOptionspopper.js 的参数object-
transition动画名称stringel-fade-in-linear
openDelay触发延迟时间number0
closeDelay关闭延迟时间number200
effect默认提供的主题stringdark
