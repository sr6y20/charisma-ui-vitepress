Contents

Text ​

文本的常见操作

基本用法 ​

由 type 属性来选择 Text 的类型。

DefaultPrimarySuccessWarningDanger
<template>
  <div class="ch-row">
    <Text>Default</Text>
    <Text type="primary">Primary</Text>
    <Text type="success">Success</Text>
    <Text type="warning">Warning</Text>
    <Text type="danger">Danger</Text>
  </div>
</template>

尺寸 ​

使用 size 属性配置尺寸，可选的尺寸大小有: lg, md 或 sm

LargeMediumSmall
<template>
  <div class="ch-row">
    <Text size="lg">Large</Text>
    <Text size="md">Medium</Text>
    <Text size="sm">Small</Text>
  </div>
</template>

省略 ​

通过 truncated 属性，在文本超过视图或最大宽度设置时展示省略符。 通过 line-clamp 属性控制多行的样式

Self element set width 100pxSqueezed by parent elementThe -webkit-line-clamp CSS property
allows limiting of the contents of
a block to the specified number of lines.
<template>
  <div class="flex-col">
    <Text truncated width="150px">Self element set width 100px</Text>
    <Text truncated width="150px">Squeezed by parent element</Text>
    <Text line-clamp="2"
      >The -webkit-line-clamp CSS property<br />
      allows limiting of the contents of<br />
      a block to the specified number of lines.</Text
    >
  </div>
</template>

覆盖 ​

使用属性 tag 覆盖元素

span

This is a paragraph.

BoldItalic This is subscript This is superscriptInsertedDeletedMarked
<template>
  <div class="flex-col">
    <Text>span</Text>
    <Text tag="p">This is a paragraph.</Text>
    <Text tag="b">Bold</Text>
    <Text tag="i">Italic</Text>
    <Text>
      This is
      <Text tag="sub" size="small">subscript</Text>
    </Text>
    <Text>
      This is
      <Text tag="sup" size="small">superscript</Text>
    </Text>
    <Text tag="ins">Inserted</Text>
    <Text tag="del">Deleted</Text>
    <Text tag="mark">Marked</Text>
  </div>
</template>

对齐 ​

使用属性 align 对齐元素

spanspanspan
<template>
  <div class="flex-col">
    <Text align="left" style="width: 100%">span</Text>
    <Text align="center" style="width: 100%">span</Text>
    <Text align="right" style="width: 100%">span</Text>
  </div>
</template>

自定义颜色 ​

使用属性 color 自定义颜色

spanspanspan
<template>
  <div class="ch-row">
    <Text color="#FFFF00">span</Text>
    <Text color="#15A4DC">span</Text>
    <Text color="#06CA64">span</Text>
  </div>
</template>
