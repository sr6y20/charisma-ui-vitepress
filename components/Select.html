Select 选择器 ​

当选项过多时，使用下拉菜单展示并选择内容。

基础用法 ​

适用广泛的基础单选 v-model 的值

<template>
  <div>
    <Select v-model="value" :options="options" placeholder="请选择"> </Select>
  </div>
</template>

<script setup lang="ts">
import { ref } from "vue";

const value = ref("");

const options = [
  {
    label: "选项1",
    value: "1",
  },
  {
    label: "选项2",
    value: "2",
  },
  {
    label: "选项3",
    value: "3",
  },
];
</script>

</script>

自定义模板 ​

您可以使用 slot 来自定义模板。

<template>
  <div>
    <Select v-model="value" :options="options" placeholder="请选择" :render-label="renderLabel" />
  </div>
</template>

<script setup lang="ts">
import { h, ref } from "vue";

const value = ref("");

const options = [
  { label: "选项1", value: "1" },
  { label: "选项2", value: "2" },
  { label: "选项3", value: "3" },
];

// eslint-disable-next-line @typescript-eslint/no-explicit-any
const renderLabel = (item: any) => {
  if (item.value === "1") {
    return h("div", { style: { color: "red" } }, item.label);
  }
  return h("span", item.label);
};
</script>

远程搜索 ​

从服务器搜索数据，输入关键字进行查找

<template>
  <div>
    <Select
      v-model="value"
      remote
      filterable
      :remoteMethod="fetchOptions"
      :options="options"
      placeholder="请选择"
    >
    </Select>
  </div>
</template>

<script setup lang="ts">
import { ref } from "vue";

const value = ref("");
const options = ref([]);

const fetchOptions = (value: string) => {
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve(
        [
          { label: "Option 1", value: "1" },
          { label: "Option 2", value: "2" },
          { label: "Option 3", value: "3" },
        ].filter((option) => option.label.includes(value))
      );
    }, 2000);
  });
};
</script>

多选 ​

适用性较广的多选场景。

<template>
  <Select v-model="values" multiple :options="options" placeholder="请选择" />
</template>

<script setup lang="ts">
import { ref } from "vue";

const values = ref<string[]>([]);
const options = [
  { label: "选项1", value: "1" },
  { label: "选项2", value: "2" },
  { label: "选项3", value: "3" },
  { label: "选项4", value: "4" },
  { label: "选项5", value: "5" },
];
</script>

Select API ​

参数说明类型默认值
multiple是否多选booleanfalse
modelValue当前选中的值string-
options选项列表SelectOption[]-
placeholder占位符string-
disabled是否禁用booleanfalse
clearable是否可清除booleanfalse
renderLabel自定义渲染标签RenderLabelFunc-
filterable是否可搜索booleanfalse
filterMethod自定义搜索方法CustomFilterFunc-
remote是否远程搜索booleanfalse
remoteMethod自定义远程搜索方法CustomFilterRemoteFunc-
